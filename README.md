# Donation & Charity Tracker System

## 📌 Description

A web application to track donations and ensure they reach verified recipients.

## 🚀 Features

* Donor registration
* Donation tracking
* Charity verification
* Secure fund distribution

## 🛠 Tech Stack

* Node.js
* Express.js
* MongoDB

## ▶️ How to Run

1. Clone the repository
2. Run: `npm install`
3. Create a `.env` file with your MongoDB URI
4. Run: `node server.js`

## ⚠️ Note

* `node_modules` is not included
* Install dependencies using `npm install`
